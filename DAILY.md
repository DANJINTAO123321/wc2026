# 每日更新工作流 (DAILY)

## 更新频率
- **每天一次**（北京时间 17:00 - 18:00 推送），遇到重大事件（伤停/天气/裁判）随时更新
- 比赛日（6/11 - 7/19）每天 2 次：08:00 早盘 + 18:00 复盘

## 数据采集源（按权重）

### 1. 国内（权重 50%）
- 腾讯体育 / 网易体育 / 新浪体育 当日赛事前瞻
- 懂球帝 / 直播吧 伤停/首发/裁判消息
- 中国体彩官方赔率 (sporttery.com.cn)
- 微博热搜体育榜、知乎体育热榜

### 2. 全球（权重 30%）
- FIFA 官方 fifa.com
- ESPN / BBC Sport / Sky Sports
- The Athletic（深度伤病/战术）
- Reuters / AP 当日赛事报道

### 3. 数据/赔率（权重 15%）
- Opta / Sofascore / FotMob
- Whoscored / Sportsmole（比分预测）
- Oddsportal（全球赔率隐含概率）
- FiveThirtyEight / Goldman Sachs（AI 概率）

### 4. 中文 AI 汇总（权重 5%，仅作参考）
- 千问 / 豆包 / Kimi / Deepseek 4 家一致意见

## 必做流程
1. 抓取 4 场（或当日所有）比赛双方近 3 场战绩 + 伤停 + 首发
2. 交叉比对 3 家以上数据源的预测比分
3. 比对中国体彩赔率隐含概率，看市场倾向
4. 综合给每场 3 个最可能比分（首推 + 次选 + 冷门）+ 各自概率
5. 更新 `index.html` 中：
   - mpred 卡片（4 场/今日）
   - ai-pred 表（4 大 AI 一致意见）
   - 比分预测底部
6. 提交 commit + push

## 概率口径
- 首推：45-55%（最有把握）
- 次选：20-30%
- 冷门：5-15%
- 4 大 AI 一致 = 加权 × 1.2（不超过 55%）
- 3/5 AI 一致 = 加权 × 1.1
- 隐含概率（中国体彩）≥ 60% = 市场强烈看好
